# Java Calculator

This is a simple calculator program written in **Java**.  
It performs basic arithmetic operations using a switch-case statement.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Uses user input for selecting the operator

## How it works
- Two numbers (`a = 10` and `b = 5`) are predefined in the program
- The user enters an operator (`+`, `-`, `*`, `/`)
- Based on the operator, the corresponding operation is performed

## Example
Enter operator (+, -, *, /): +
Addition: 15.0
